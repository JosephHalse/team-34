Recognition of Medical Equipment for Smart Glasses 

Our client NTT Data, asked us to make an application to demonstrate their capabilites and gave us a use case of medical professoinals needing hands-free technology. Therefore we designed an application based upon Epson Smart Glasses. The user wears the smart glasses, that by use of the inbuilt camera, will “see” and recognise various medical equipment, label them and display information about their use. This gives the user aid in identifying the item, understanding its purpose and how to use it. Instructions are further displayed as text on the glasses, allowing the user to keep their hands free from printed manuals, saving time. 

In our final prototype we achieved this by using Fritz API for performing real time object recognition for the app and TensorFlow to create machine learning models of items to be recognised.

Key Features

Real time object recognition 

The camera is able to recognize multiple objects in real time and identify where they are in a 3D environment.

Ability to display information 

Allows for customised instructions of objects to be displayed on the glasses.

Voice Recognition 

Quick and responsive voice recognition is used in order to navigate our application.
