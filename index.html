Recognition of Medical Equipment with Augmented Reality for Smart Glasses 

This application is designed to be used by medical staff and amateur first aiders to help with the use of medical equipment by using Augmented Reality displayed on Epson Smart Glasses. The user wears the smart glasses, that by use if the inbuilt camera, will “see” and recognise various items of medical equipment and display information about their use. This gives the user aid in identifying the item, understanding its purpose and easily accessing instructions as to how to use it, which is displayed as text on the glasses without need to reference printed paper manuals, keeping hands free and saving time. 

We have set about achieving this by using the Moverio AR SDK, built specifically for the glasses, taking photogrammetry scans of real life objects (so far this includes some basic medical equipment such as blood pressure monitor, digital thermometer) and used these to make 3D models and training data for use by the application.

Key Features

Real time object recognition 

Identifies data from the camera as the location of an object in 3D space.

Ability to display information 

Allows for customised instructions to be delivered for specific objects.

Interactive User Experience: 

The user interface is ergonomically designed for maximum ease of use while maintaining high fidelity and fluidity
