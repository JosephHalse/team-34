Prototype

So far we have only managed to make a basic prototype which is programmed to recognise two sample test objects: medical equipment consisting of: 1) a blood pressure monitor and: 2) an infrared digital thermometer. This was achieved using the Epson Moverio Software Development Kit (SDK), compatible with the Epson Smart Glasses and also with Android devices.
3D models of the equipment to be identified were created using photogrametry, which involved taking between 60 and 100 photographs of that item in a lighting environment that reduces shadows.  This are then processed with photogrammetry software and the 3D model is generated.
This is then used to train the recognition model which is then implimented into the application.
When viewed on the smart glasses there are two modes which can be selected with the hand held controller: 2D overlay mode where it apears as if on a screen in front of you and 3D Augmented Reality mode where the text looks like it's really there, floating above the object it refers to.
